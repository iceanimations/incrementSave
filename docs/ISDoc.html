What is IncrementalSave?

IncrementalSave makes it easy to save your Maya File or Scene as increments. So you don't need to save your scene file manually by changing the increment number every time you make changes to your scene. When using IncrementalSave, you need to save your scene only once in usual way, my_scene_file001.mb for example. When a file is saved and you make changes to it then you just have to run IncrementalSave with only one click and your current scene file will be saved as my_scene_file002.mb in the same directory in which the my_scene_file001 lives. Next time you run this tool, it will save the current scene as my_scene_file003.mb and so on.

How to access IncrementalSave?

This tool is available under ICE Scripts menu inside Maya. Go to ICE Scripts > General > Save_Increment. If you click Save_Increment, the file will be saved, so be careful when clicking on the tool.
